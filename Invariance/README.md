## 有图必有谱

谱聚类是针对“图”的节点分割算法。
理想的谱聚类需要解决 NP-Hard 问题：
分割导致解空间不连续；
而节点搜索导致计算规模随着数据规模而扩大。
因此，谱聚类算法是寻找一个可行的近似解。
