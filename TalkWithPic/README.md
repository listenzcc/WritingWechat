## 降维打击-四元数

用四维的形式解决三维的问题。

在四元数规范下，
我们将看到内积和外积对应同一个方程的实部和虚部，
也因此，三维世界的解析几何经典公式在四维世界中成对出现。

这也是目前主要的计算机视觉工具在处理三维空间几何计算问题时，
使用四元数对空间坐标进行表示的原因。
