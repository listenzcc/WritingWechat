---## SORA的盲人摸象

从目前的消息来看，SORA是一个先进的视频生成模型，它通过变分自编码器（VAE）将视频编码为隐空间中的时空patches，然后利用基于Transformer的扩散模型对这些patches进行操作，以生成高质量的长视频内容。
